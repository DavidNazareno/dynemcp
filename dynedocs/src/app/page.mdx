## Key Features

### 🚀 Fast Development

Hot reload and instant feedback for rapid development cycles.

### 🏗️ Optimized Build System

Ultra-fast builds with esbuild and modern tooling stack.

### 📝 Declarative Configuration

Simple JSON-based configuration that's easy to understand and maintain.

### 🌐 Multiple Transports

Support for stdio, HTTP, SSE, and more transport protocols.

### 🔧 Dynamic Registry

Runtime registration of tools, resources, and prompts for flexible architectures.

### 🎯 Model Sampling

Built-in LLM model sampling support for AI-powered applications.

### 🔒 Security Ready

Authentication, rate limiting, and CORS out of the box.

## Quick Start

```bash
# Install globally
npm install -g @dynemcp/create-dynemcp

# Create new project
create-dynemcp my-agent

# Start development
cd my-agent
npm run dev
```
