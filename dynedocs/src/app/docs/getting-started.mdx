# Getting Started

Welcome to DyneMCP! This guide will help you get up and running with the Model Context Protocol (MCP) framework in just a few minutes.

## What is DyneMCP?

DyneMCP is a comprehensive framework for building MCP (Model Context Protocol) servers. It provides everything you need to create, develop, and deploy MCP servers with modern TypeScript development tools.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js 18+**: [Download from nodejs.org](https://nodejs.org/)
- **pnpm** (recommended): `npm install -g pnpm`

## Quick Start

## Global Installation

```bash
npm install -g @dynemcp/create-dynemcp
```

## Create a New Project

```bash
# Interactive mode
create-dynemcp my-agent

# With specific template
create-dynemcp my-calculator --template calculator

# Non-interactive mode
create-dynemcp my-agent --yes --template default
```

## Local Development

```bash
# Clone repository
git clone https://github.com/dynemcp/dynemcp.git
cd dynemcp

# Install dependencies
pnpm install

# Run all tests
pnpm test

# Build all packages
pnpm build

# Development with watch mode
pnpm dev
```

## Creating Your First MCP Server

Let's create a simple MCP server with a basic tool:

### 1. Create Configuration

Create a `dynemcp.config.json` file:

```json
{
  "name": "my-first-server",
  "version": "1.0.0",
  "description": "My first MCP server",
  "transport": {
    "type": "stdio"
  },
  "build": {
    "outDir": "dist",
    "bundle": true
  }
}
```

### 2. Create a Simple Tool

Create `src/tools/greeting.ts`:

```typescript
import { DyneMCPTool } from '@dynemcp/dynemcp'
import { z } from 'zod'

const GreetingSchema = z.object({
  name: z.string().describe('Name of the person to greet'),
})

export class GreetingTool extends DyneMCPTool {
  get name() {
    return 'greet'
  }

  readonly description = 'Greets a person by name'
  readonly schema = GreetingSchema

  async execute(input: z.infer<typeof GreetingSchema>) {
    const { name } = input
    return {
      greeting: `Hello, ${name}! Welcome to DyneMCP!`,
      timestamp: new Date().toISOString(),
    }
  }
}

export default new GreetingTool()
```

### 3. Create Main Server File

Create `src/index.ts`:

```typescript
import { createMCPServer } from '@dynemcp/dynemcp'

async function main() {
  const server = createMCPServer(
    'my-first-server',
    './dynemcp.config.json',
    '1.0.0'
  )

  await server.init()
  await server.start()

  console.log('MCP Server started successfully!')
}

main().catch(console.error)
```

### 4. Add Package Scripts

Update your `package.json`:

```json
{
  "scripts": {
    "dev": "dynemcp dev",
    "build": "dynemcp build",
    "start": "node dist/index.js"
  }
}
```

## Running Your Server

### Development Mode

Start your server in development mode with hot reload:

```bash
npm run dev
```

This will:

- Watch for file changes
- Automatically rebuild
- Restart the server
- Show detailed logs

### Production Build

Build your server for production:

```bash
npm run build
```

This creates optimized bundles in the `dist` directory.

### Production Start

Run your built server:

```bash
npm start
```
