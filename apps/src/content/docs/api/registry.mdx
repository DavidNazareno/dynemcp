# Registry API

The registry lets you load and query all your tools, resources, and prompts in one place.

## Usage Example

```typescript
import { registry } from '@dynemcp/dynemcp'

await registry.loadAll({
  tools: { enabled: true, directory: './src/tools' },
  resources: { enabled: true, directory: './src/resources' },
  prompts: { enabled: true, directory: './src/prompts' },
})

const allTools = registry.getAllTools()
const tool = registry.getTool('my-tool')
```

- Use `getAllTools`, `getTool`, `getAllResources`, `getResource`, `getAllPrompts`, `getPrompt` to access your components.
- No need to manage loading or caching yourself.
